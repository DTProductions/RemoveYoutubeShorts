# Remove YouTube Shorts

This is an open-source Chrome extension that removes YouTube Shorts from YouTube.

## Features

- Remove the Shorts tab on YouTube's start page.
- Remove all short recommendations from the start page, search page, and normal video's sidebar.
- Remove shorts from channels' shorts tab.
- Redirect to YouTube's start page if trying to load a short (such as via a direct URL or clicking on a search engine recommendation).
